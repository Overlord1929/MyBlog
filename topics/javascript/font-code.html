Controlando vídeos com javascript

10/01/2018

Neste artigo explicarei como controlar videos (criando controles) usando funções javascript! Primeiro vamos adicionar nosso vídeo:

Play
Pause
Bigger
Normal
Smaller


Para isso, utilizamos a tag <video> com a subtag <source>:



<video id="myVideo">
    <source src="" type="">
< /video>


A seguir cria-se os botões que receberão as funções de controle do vídeo.


<button class="video-button">Play</button>
<button class="video-button">Pause</button>
<button class="video-button">Bigger</button>
<button class="video-button">Normal</button>
<button class="video-button">Smaller</button>

Utiliza-se o atributo onclick de cada botão que vai acionar a função específica dele...
